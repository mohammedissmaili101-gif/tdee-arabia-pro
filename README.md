# TDEE ARABIA 🔥
أدق حاسبة سعرات حرارية وماكروز للسوق العربي.

## المميزات:
- **حساب دقيق:** يعتمد معادلة Mifflin-St Jeor العالمية.
- **توليد تلقائي:** مقالات SEO يومية باستخدام AI (Llama 3).
- **تصميم ملاكي:** متوافق مع جميع الشاشات وبسرعة خيالية.

## التقنيات المستخدمة:
- HTML5 / Tailwind CSS
- Python (Content Generator)
- GitHub Actions (Automation)
- Vercel (Hosting)
